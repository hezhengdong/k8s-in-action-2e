# 第四部分 连接与暴露应用

第四部分聚焦于应用之间的通信——无论是集群内部的应用之间，还是与外部客户端的通信。

在第 11 章中，你将学习 Pod 之间如何相互通信。你将使用 Service 对象将一组提供相同功能的 Pod 暴露在单个稳定的地址下，并了解其他 Pod 如何发现和使用该地址。你还将学习如何将服务暴露给外部客户端，以及 Pod 如何向 Kubernetes 发出信号表明自己是否已准备好接收流量。

第 12 章在此基础上讲解如何使用 Ingress 对象通过单个公共 IP 地址将多个服务暴露给外部客户端。

第 13 章涉及类似的主题，但引入了 Gateway API 作为 Ingress 的替代方案。你将了解 Gateway API 如何使用 Gateway、HTTPRoute、TLSRoute、TCPRoute、UDPRoute 等对象来暴露内部或外部的服务。
