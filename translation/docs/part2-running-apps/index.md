# 第二部分 在 Kubernetes 中运行应用

本书第二部分深入探讨在 Kubernetes 中运行、组织应用并保持其健康的核心概念。

第 5 章介绍 Pod 对象——Kubernetes 的基本构建块。你将看到应用如何在 Pod 内的容器中运行，以及 Pod 如何包含额外的辅助容器来支持或增强应用的功能。

第 6 章探讨 Pod 在整个生命周期中发生的事情。你将学习如何在容器启动或停止时执行操作，如何保持主容器和辅助容器的健康，以及 Kubernetes 如何管理就绪、存活和启动状态。

在第 7 章中，你将学习如何组织大量的 Pod 和其他 Kubernetes 对象，使其用途和所属关系始终清晰，从而让你的集群更易于理解和维护——无论是对你还是对你的团队成员。
